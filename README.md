# fastq-dmux
FASTQ demultiplexer

## Usage
    Usage: fastq-dmux --conditions <path> --fastq <path> [--output-dir <path>]
    
    Demultiplexes FASTQ files based on conditions
    
    Options and flags:
        --help
            Display this help text.
        --conditions <path>
            The conditions file
        --fastq <path>
            The FASTQ file
        --output-dir <path>, -o <path>
            The output directory
